$<$/details$>$\hypertarget{md__home_geochip_Dev_FloorplanToBlender3d_git_refactoring_README_autotoc_md0}{}\doxysection{About}\label{md__home_geochip_Dev_FloorplanToBlender3d_git_refactoring_README_autotoc_md0}
The virtualisation of real life objects has been a hot topic for several years. As I started learning about 3d modelling in \href{https://www.blender.org/}{\texttt{ Blender3d}} I thought of the idea to use simple imaging on floorplans to automatically create corresponding 3d models. It is much easier than it sounds and uses a low amount of resources, enabling it to be used on low hardware. By utilizing Blender3d, all created objects will be easy to transfer to any other 3d rendering program. Such as \href{https://unity.com/}{\texttt{ Unity}}, \href{https://www.unrealengine.com/en-US/}{\texttt{ Unreal Engine}} or \href{https://www.autodesk.com/solutions/cad-software}{\texttt{ CAD}}.\hypertarget{md__home_geochip_Dev_FloorplanToBlender3d_git_refactoring_README_autotoc_md1}{}\doxysection{How-\/\+To}\label{md__home_geochip_Dev_FloorplanToBlender3d_git_refactoring_README_autotoc_md1}

Firstly you need to install a suitable \href{https://www.docker.com/}{\texttt{ Docker}} environment on your device. This project contains a {\ttfamily Docker\+File} which uses the {\ttfamily Ubuntu 18.\+04} image so make sure your docker environment is set to linux containers.

This project is linked to \href{https://hub.docker.com/r/grebtsew/floorplan-to-blender}{\texttt{ Docker Hub}} which means a maintained and prebuilt container can be pulled directly by running\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{docker pull grebtsew/floorplan-\/to-\/blender}
\DoxyCodeLine{\# run the container:}
\DoxyCodeLine{docker run -\/it grebtsew/floorplan-\/to-\/blender}

\end{DoxyCode}


{\bfseries{NOTE}} \+: For more information about how the dockerfile and docker-\/compose files can be used to build and run the image to add your own content read more here.

To pull and run the container together in a one line command run\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{docker-\/compose run ftb}

\end{DoxyCode}
